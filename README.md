🎓 Student Depression Predictor

📌 Overview
The Student Depression Predictor is a machine learning project aimed at predicting student depression based on various factors. The project involves data preprocessing, feature engineering, model training, and evaluation to build an effective classification model.

🚀 Features
✅ Exploratory Data Analysis (EDA) to extract insights from the dataset.
✅ Data Preprocessing including missing value handling, encoding,checking multicoinearty and transformation.
✅ PowerTransformer Application to normalize numerical values.
✅ Multiple Machine Learning Models including Logistic Regression, Decision Tree, Random Forest, AdaBoost, and XGBoost.
✅ Hyperparameter Tuning for optimizing model performance.
✅ Final Model Selection based on accuracy, precision, recall, and F1-score.

📂 Dataset
The dataset contains multiple student-related attributes that contribute to the prediction of depression. Key features include:

Demographic Details (Age, Gender, etc.)

Academic & Social Factors

Mental Health & Family History

🛠️ Tech Stack
Programming Language: Python 🐍

Libraries: Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn

Algorithms Used: Logistic Regression, Decision Tree, Random Forest, SVM, AdaBoost, XGBoost
